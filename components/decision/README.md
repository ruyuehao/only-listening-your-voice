# Decision FSM + LED Module (决策调度 + 状态指示)

## 概述

有限状态机驱动的系统调度器，配合板载 WS2812 RGB LED 提供实时状态指示。

## 状态机

```
                    ┌─────────────────────────────┐
                    │         STATE_IDLE           │
                    │     绿灯 1Hz 闪烁            │
                    └──────────┬──────────────────┘
                               │ EVENT_KWS_TRIGGERED
                    ┌──────────▼──────────────────┐
                    │      STATE_VERIFYING         │
                    │     绿灯 2Hz 快闪            │
                    └──────────┬──────────────────┘
                               │ SV 推理完成
                    ┌──────────▼──────────────────┐
               ┌────┤  相似度 ≥ 0.70 ?            │
               │    └─────────────────────────────┘
               │ YES                    │ NO
    ┌──────────▼──────────┐  ┌──────────▼──────────┐
    │   STATE_ACCEPTED    │  │   STATE_REJECTED    │
    │   红灯常亮 1s       │  │   红灯闪烁 1.5s     │
    └──────────┬──────────┘  └──────────┬──────────┘
               │                        │
               └────────┬───────────────┘
                        │ 超时
               ┌────────▼──────────┐
               │    STATE_IDLE     │
               └───────────────────┘
```

## LED 颜色方案

| 状态 | 颜色 | 模式 |
|------|------|------|
| IDLE | 绿色 | 1Hz 闪烁 (500ms on/off) |
| LISTENING | 绿色 | 常亮 |
| VERIFYING | 绿色 | 2Hz 快闪 (250ms on/off) |
| ACCEPTED | 红色 | 常亮 1 秒 |
| REJECTED | 红色 | 5Hz 闪烁 (100ms on/off) |
| ENROLL | 蓝色 | 2Hz 快闪 |

## 文件

| 文件 | 说明 |
|------|------|
| `fsm.h/c` | 有限状态机核心 |
| `led_indicator.h/c` | WS2812 LED 驱动 (led_strip) |
| `task_decision.h/c` | 决策调度 FreeRTOS 任务 |

## 依赖

- `espressif/led_strip` (>= 3.0.0) — RMT 驱动 WS2812
- FreeRTOS Event Groups
